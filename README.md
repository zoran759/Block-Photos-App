## Testing the project

To test this project do the following...

Installation:
```bash
git clone https://github.com/nerdic-coder/block-photos.git block-photos
cd block-photos
npm install
```

Starting Electron app:
```bash
npm start
```

Build and package Electron app:
```bash
npm run package
```

Test ESLint:
```bash
npm run lint
```

Run unit tests:
```bash
npm run test
```
